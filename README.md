#Cách Dùng:
  - Thêm VICTORY.zip vào resource_packs và Chỉnh 
resource_packs.yml thành:

      force_resources: true
      resource_stack:
      - VICTORY.zip
  
    
  - Bạn có Thể tùy chỉnh số tiền bán item tại config.yml
